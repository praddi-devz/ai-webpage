# AI and Its Real-World Applications

Hi!

This is my Computer project on **Artificial Intelligence (AI) and Its Real-World Applications**.

I made this website using **HTML and CSS**. It explains what AI is, where it is used in our daily lives, its advantages and disadvantages, and some interesting facts about AI.

The website also has a separate **History of AI** page which shows how AI has developed over the years with images and information.

## Files Included

- index.html (Main page)
- history.html (History of AI page)
- Images used in the website

## Features

- Simple and easy-to-use design
- Navigation bar
- AI applications
- Quick facts
- Interesting facts
- History of AI
- Images and summary table
